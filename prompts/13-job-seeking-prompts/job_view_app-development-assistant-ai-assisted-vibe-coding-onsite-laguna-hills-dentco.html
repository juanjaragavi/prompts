About the Role

Onsite App Development Assistant at Dentco building web and mobile apps using AI-assisted coding tools to rapidly prototype and ship products. Work directly with the owner in Laguna Hills, CA to implement app concepts, handle hosting/deployment, integrations, and clean up AI-generated code.

Job Description

Role

An onsite App Development Assistant working directly with the company owner to turn app and SaaS concepts into working prototypes and shipped products. The role emphasizes AI-assisted, rapid prototyping and hands-on implementation rather than a traditional senior engineering position.

Key Responsibilities

  • Build web and mobile apps using AI-assisted coding workflows and iterate quickly on feedback.
  • Handle technical plumbing: hosting and deployment, domains, databases, user authentication, and app store submissions.
  • Debug and clean up AI-generated code to maintain project quality and maintainability.
  • Set up integrations such as payments (Stripe), email/SMS, and basic APIs.
  • Keep projects organized in Git and document implementations.
  • Research tools and recommend the fastest, cheapest ways to test ideas.
  • Work onsite with the owner and follow product direction.

Requirements

  • Real coding experience with at least one end-to-end shipped app or website (personal projects and freelance work count).
  • Working knowledge of JavaScript or TypeScript and at least one modern framework (examples: React, Next.js, React Native, Flutter).
  • Hands-on experience with AI coding tools and prompt-driven development workflows.
  • Ability to follow direction from the product owner and execute their vision.
  • Comfortable working with ambiguity and turning rough sketches into working prototypes.
  • Must be able to work onsite at the Laguna Hills, CA location.

Nice to Have

  • Experience launching apps to the Apple App Store and/or Google Play.
  • Basic UI/UX design sense and familiarity with Figma or equivalents.
  • Experience with SaaS patterns like subscriptions, user accounts, and admin dashboards.
  • Familiarity with no-code/low-code tools (Bubble, Webflow, Zapier/Make).
  • Interest in the automotive space.

Compensation & Location

  • Pay: $20.00 - $40.00 per hour (or equity as an option mentioned).
  • Work Location: In person, Laguna Hills, CA.
  • Role may be offered as Full-Time or Part-Time / In-House.

How to Apply

  • Provide a link to something you’ve built (live app, GitHub, or demo video).
  • Include 2–3 sentences on how you use AI tools in your coding workflow.
  • State your availability and rate expectations.
  • Applications without a portfolio link will not be considered.
