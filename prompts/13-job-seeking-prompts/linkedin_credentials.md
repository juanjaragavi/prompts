# LinkedIn Credentials and API Keys Retrieval

Credential values have been removed from source control. Store real values in a local `.env` file or a secrets manager.

- **LINKEDIN_CLIENT_ID**: `YOUR_LINKEDIN_CLIENT_ID`
- **LINKEDIN_CLIENT_SECRET**: `YOUR_LINKEDIN_CLIENT_SECRET`
- **LINKEDIN_ACCESS_TOKEN**: `YOUR_LINKEDIN_ACCESS_TOKEN`
- **LINKEDIN_REFRESH_TOKEN**: `YOUR_LINKEDIN_REFRESH_TOKEN_OR_NOT_APPLICABLE`
